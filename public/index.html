      Images
      
            
      
              Default export is WebP, which usually gives the best size and
              quality balance for the web. Use AVIF for even smaller files on
              modern browsers. SVGs are rasterized when converting to pixel
              formats.
            
      
          
      
          
      
            
      Video
      
            
      
              WebM uses VP9 and Opus. Video compression needs
              FFmpeg on your Mac (e.g.
              brew install ffmpeg), then restart the app. If the
              app still cannot find it, set FFMPEG_PATH to your
              ffmpeg binary (often
              /opt/homebrew/bin/ffmpeg). Optional max width scales
              video down; bitrate overrides quality-based encoding for WebM when
              set.
