      Tips
      
          
      
            Three quick things people ask about. Still stuck? Read the short
            note under each topic.
          
      
          
      
            
      
              
      Photos
      
              
      
                WebP is usually the sweet spot: small file, still
                looks great on a screen. AVIF can be even
                smaller on newer phones and browsers. Fancy vector
                SVG files turn into regular pictures if you pick
                JPEG or WebP.
              
      
            
      
            
      
              
      Video
      
              
      
                Shrinking video needs a helper app called
                FFmpeg on the computer that runs this site. On a
                Mac you can install it with
                brew install ffmpeg, then start the site again. If
                it still says it cannot find FFmpeg, point
                FFMPEG_PATH to where it lives, often
                /opt/homebrew/bin/ffmpeg.
